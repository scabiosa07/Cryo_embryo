# 🧪 Virtual Lab: Cryopreservation of Embryos  
  
## Description  
This project simulates embryo cryopreservation using Streamlit.  
  
## Features  
- Embryo quality selection  
- Cryoprotectant simulation  
- Freezing process  
- Thawing and survival prediction  
  
## How to Run  
```bash  
pip install -r requirements.txt  
streamlit run app.py
